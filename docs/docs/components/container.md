## Width

x = screen width

|                 | x < 600px | 600px <= x < 1000px | x >= 1000px |
| :-------------: | :-------: | :-----------------: | :---------: |
| container width |    92%    |         84%         |     76%     |

You can change container width in the [config](../config#container).
