# Get started

<div class="mb--10"></div>

## Via CDN

Place the following link in the head of your html file.

::: code-group

<!-- prettier-ignore -->
```html [jsDelivr]
<link href="https://cdn.jsdelivr.net/npm/csslandia/style.min.css" rel="stylesheet" />
```

```html [unpkg]
<link href="https://unpkg.com/csslandia/style.min.css" rel="stylesheet" />
```

:::

## Via Package Managers

1. Install Csslandia

::: code-group

```sh [npm]
npm i csslandia
```

```sh [Yarn]
yarn add csslandia
```

```sh [pnpm]
pnpm add csslandia
```

:::

2. Import Csslandia

::: code-group

```js [JS]
import "csslandia";
```

```html [HTML]
<head>
    <link rel="stylesheet" href="node_modules/csslandia/style.min.css" />
</head>
```

```css [CSS]
@import "csslandia";
```

:::
